#### Language Used: Go




## Run Locally

Clone the project

```bash
  git clone git@github.com:nilesh304/fetch-nilesh.git
```

Go to the project directory

```bash
  cd fetch-nilesh
```

Install dependencies

```bash
  go mod tidy
```

To Start the server

#### Method 1

```bash
  go run main.go
```


#### Method 2

```bash
  go build -o receipt-processor
```
```bash
  ./reciept-processor
```
